# Функция выбора шрифтов для текстовых объектов

## Описание
Добавлена возможность выбора различных шрифтов для текстовых объектов (ObjectType_Text) в редакторе объектов Metasiberia.

## Установка шрифтов

1. **Распаковать архив:**
   - Файл: `C:\programming\fonts.rar`
   - Распаковать в: `C:\programming\substrata\resources\fonts\`
   - В этой папке должны находиться файлы шрифтов (*.ttf, *.otf, *.fon)

2. **Структура:**
   ```
   C:\programming\substrata\resources\fonts\
   ├── Arial.ttf
   ├── Times.ttf
   ├── Courier.fon
   └── ... другие шрифты
   ```

## Использование

### В редакторе объектов:
1. Создайте текстовый объект через меню "Add Text"
2. Выберите объект в 3D сцене
3. В панели "Editor" найдите поле "Font" (расположено между "Content" и "Target URL")
4. Выберите нужный шрифт из выпадающего списка
5. Текст будет отрисовываться выбранным шрифтом

### Структура UI редактора:
```
Content:     [QTextEdit for text content]
Font:        [QComboBox with font selection] ← НОВОЕ
Target URL:  [QLineEdit]
```

## Как это работает

### Фронтенд (GUI клиент):
- ComboBox автоматически заполняется при инициализации редактора
- Список загружается из папки `resources/fonts/`
- Поддерживаются форматы: `.ttf`, `.otf`, `.fon`
- При изменении шрифта объект помечается как измененный и отправляется на сервер

### Бэкенд (WorldObject):
- Новое поле: `std::string text_font` - имя выбранного шрифта
- Флаг изменения: `TEXT_FONT_CHANGED = 128`
- Сохраняется в XML (при сохранении объектов)
- Передается по сети (writeToNetworkStream)
- Сохраняется в базе данных (при отправке на сервер)

## Реализованные изменения

### ObjectEditor.ui
- Добавлена label "Font" (row 16, column 0)
- Добавлена QComboBox "fontComboBox" (row 16, column 1)

### ObjectEditor.cpp
- `loadAvailableFonts()` - загружает доступные шрифты из папки
- `setFromObject()` - устанавливает текущий шрифт при открытии редактора
- `toObject()` - сохраняет выбранный шрифт при изменении
- `setControlsEditable()` - управляет доступностью ComboBox

### WorldObject
- Новое поле состояния: `text_font`
- Константа для ограничения размера: `MAX_FONT_NAME_SIZE = 256`
- Флаг для отслеживания изменений: `TEXT_FONT_CHANGED`
- Сеарилизация/десеарилизация в сетевой поток
- XML сеарилизация/десеарилизация

## Дополнительная интеграция (не реализовано в этом обновлении)

Для полной функциональности требуется обновить код отрисовки текста:
- Файл отрисовки текста в 3D (например, в WorldObject rendering code)
- TextRenderer должен использовать `text_font` параметр вместо default
- Загрузка шрифтов в памяти GPU при необходимости

## Примечания

- "Default" - зарезервированное имя для стандартного шрифта (добавляется первым в список)
- Если шрифт не найден или неверно указан, используется "Default"
- Максимальная длина имени шрифта: 256 символов
- Шрифты автоматически загружаются при инициализации редактора

## Совместимость

- Добавлено как новое поле в `text_font`
- Не ломает совместимость с существующими объектами (используется Default для старых)
- Поддерживается как сохранение в XML, так и передача по сети
