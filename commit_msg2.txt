Добавлено автоматическое копирование иконок кнопок при сборке

Изменения:
- Добавлена POST_BUILD команда в CMakeLists.txt для автоматического копирования иконок из resources/buttons/ в выходную директорию
- Теперь иконки автоматически копируются при каждой сборке проекта

Преимущества:
- Не требуется ручное копирование файлов иконок
- Иконки обновляются автоматически при каждой сборке
- Работает для всех конфигураций (Release, RelWithDebInfo, Debug)

Как использовать:
1. Замените файлы иконок в C:\programming\substrata\resources\buttons\
2. Пересоберите проект - иконки автоматически скопируются в нужное место
3. Перезапустите приложение для применения изменений
