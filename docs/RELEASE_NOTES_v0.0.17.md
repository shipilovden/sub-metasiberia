# Metasiberia Beta v0.0.17

Дата релиза: 2026-03-25

## Что вошло в релиз

1. Исправлена клиентская регрессия текстовых объектов: редактор больше не отправляет text-объекты в generic mesh-loader с пустым `model_url`, поэтому создание и редактирование текста снова работает без ошибки `Format not supported`.
2. Исправлено мгновенное применение выбранного шрифта для 3D-текста: клиент теперь реально разрешает `WorldObject::text_font` в файл из `data/resources/fonts`, кэширует соответствующий набор шрифтов и пересобирает геометрию текста сразу после смены шрифта.
3. Устранена reentrant-регрессия в `ObjectEditor`: во время `setFromObject()` блокируются сигналы редактора, а для не-модельных объектов больше не перезаписывается `model_url`, что стабилизирует редактирование text/web/audio-объектов.
4. Исправлена логика seek в аудиоплеере для webview/audio panels: CEF resource handler теперь корректно обрабатывает начальный `Skip()` для аудио, а JavaScript-плеер больше не откатывает бегунок в начало сразу после клика по полосе прогресса.
5. Обновлена внутренняя документация по выбору шрифтов, чтобы зафиксировать runtime-путь применения `text_font` и недавние правки editor pipeline.

- Версия Metasiberia: `0.0.17` (`shared/MetasiberiaVersion.h`).
