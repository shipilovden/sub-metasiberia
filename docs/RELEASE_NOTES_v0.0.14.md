# Metasiberia Beta v0.0.14

Дата релиза: 2026-03-10

## Что вошло в релиз

1. Добавлены настройки тумана мира в `World Settings`: теперь доступны два слоя тумана с отдельными параметрами `thickness` и `height scale`.
2. Изменения `World Settings` применяются локально сразу, без обязательного client-server round-trip перед предпросмотром. Это ускоряет настройку солнца, воды, террейна и тумана.
3. Сервер теперь сериализует и сохраняет fog settings внутри `WorldSettings`, поэтому настройки тумана не слетают после `Apply` и переживают повторное подключение к миру.
4. В `WorldSettingsUpdate` добавлен `sender_avatar_UID`, чтобы клиент игнорировал эхо собственных изменений и не перезагружал terrain/UI повторно после локального изменения настроек мира.
5. Исправлен рендер тумана в клиенте: fog теперь действительно влияет на средние и дальние дистанции, а вычисление слоя тумана учитывает высоту относительно камеры.
6. Туман по умолчанию выключен. Он включается и настраивается только через `World Settings`, что соответствует ожидаемому поведению в upstream-логике `WorldSettings`.
7. README репозитория доработан под публичный релиз: добавлены `shields.io`-бейджи, быстрые ссылки на сайт, админку, регистрацию и основные проектные ресурсы.
8. Запущен первый структурированный блок GitHub Wiki на русском языке: стартовые страницы `01-03`, image registry, publish-скрипт и русифицированные служебные страницы wiki.
