# Metasiberia Beta v0.0.13

Дата релиза: 2026-03-04

## Что вошло в релиз

1. Улучшен выход из транспорта с учётом стороны сиденья и корректной наследуемой скорости (`vehicle exit logic`).
2. Добавлена физика плавучести и водного сопротивления для игрока под водой (`buoyancy + water drag`).
3. `PhotoModeUI` и `GestureManagerUI` переведены на оконную модель с корректным закрытием через window-event.
4. Добавлены настройки солнца мира и `terrain_height_scale` (масштаб высоты террейна).
5. Доработана синхронизация закрытия `GestureManager` и состояния кнопки `PhotoMode`.
6. В `EmbeddedBrowser` добавлен безопасный fallback при ошибках shared GPU textures.
7. Для `browser_process` добавлены экспортируемые флаги приоритета дискретного GPU.
8. Отключён шумный диагностический вывод скачивания ресурсов.
9. Отключено удаление большей части sun-data, так как направление солнца теперь может меняться по миру.
10. Обновлены CI workflow-файлы в репозитории.

## Технические примечания

- Версия клиента: `0.0.13` (`shared/Version.h`).
- Версия Metasiberia: `0.0.13` (`shared/MetasiberiaVersion.h`).
- Целевой артефакт релиза: `MetasiberiaBeta-Setup-v0.0.13.exe`.
